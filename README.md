# DPD-Filters-Keysight-Internship
A collection of the code that I wrote from scratch during my 6-month research internship in the Quantum Engineering Solutions (QES) Team in Keysight at Edinburgh.
The main aim was investigating how DSP techniques, namely DPD filters, can be utilised to correct for the dynamical distortions along flux control lines for superconducting qubits.
